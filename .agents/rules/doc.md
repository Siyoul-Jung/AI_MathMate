# doc: AI MathMate Documentation Standards

## Project Documentation
- **README.md**: Maintain technical stack and "Quick Start" instructions. 
- **Change Logs**: Document significant changes to the DB schema or LLM prompts.

## Mathematical Content
- **Templates**: Problem templates (`template.md`) should clearly separate the problem statement, solution, and DNA metadata.
- **Reports**: Comparison reports (AIME Comparison) should use tables to show variant differences.

## Inline Comments
- **Python**: Use docstrings for complex generation logic in `amc_engine`.
- **React**: Comment on complex coordinate math for SVG/Graphing components.
