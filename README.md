# AI_MathMate Project

Next.js와 Python(FastAPI)을 활용한 수학 문제 자동 생성 및 워크북 관리 플랫폼입니다.

## 📂 프로젝트 구조

- **frontend/**: Next.js 기반의 웹 프론트엔드
- **data/**: 수학 문제 데이터 (JSON 형식)

## 🛠 기술 스택

- **Framework**: Next.js 16
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **Math Rendering**: KaTeX, React-KaTeX
- **Development**: AI-Assisted with Google Gemini (Vibe Coding Methodology)

## 🚀 시작하기 (Getting Started)

이 프로젝트를 로컬에서 실행하려면 다음 단계가 필요합니다.

### 1. 저장소 클론

```bash
git clone https://github.com/Siyoul-Jung/AI_MathMate.git
cd AI_MathMate
```

### 2. 의존성 설치 및 실행

프론트엔드 디렉터리로 이동하여 패키지를 설치하고 개발 서버를 실행합니다.

```bash
cd frontend
npm install
npm run dev
```

브라우저에서 `http://localhost:3000`을 열어 확인합니다.
