# Mathematical DNA
- **Topic**: Functions & Algebra
- **Sub-topic**: Piecewise functions, periodicity, parabola intersections
- **Difficulty Pattern**: Advanced (Level 3)
- **Tag**: `FUNC-PIECEWISE-PARABOLA`

# Structure
The problem asks for the sum of the $y$-coordinates of all intersection points between a periodic sawtooth function $f(x)$ and a forward-opening parabola $x = Ky^2$. The result evaluates to an exact format $\frac{a+b\sqrt{c}}{d}$.
